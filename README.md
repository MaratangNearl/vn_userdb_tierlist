# VNUT - Visual Novel Tier List (비주얼 노벨 티어리스트)

![VNUT Icon](assets/icon.ico)

VNUT is a desktop application designed to easily create, manage, and share your Visual Novel tier lists.
VNUT은 비주얼 노벨 티어리스트를 손쉽게 작성하고 관리하며 공유할 수 있도록 설계된 데스크톱 애플리케이션입니다.

## [KOR] 주요 기능 및 사용 방법

### 🚀 주요 기능
- **VNDB API 연동**: 제목이나 ID만으로 게임 정보와 커버 이미지를 자동으로 불러옵니다.
- **다국어 지원**: 5개 국어 UI를 지원하며 설정에서 자유롭게 변경 가능합니다.
- **커스텀 디자인**: 다크/라이트 모드 지원 및 티어별 색상, 배경색을 직접 지정할 수 있습니다.
- **이미지 내보내기**: 작성한 티어표를 고화질 JPEG 이미지로 저장하여 공유할 수 있습니다.
- **데이터 관리**: 백업 및 복원 기능을 통해 데이터를 안전하게 관리합니다.

### 🔘 버튼 설명
- **추가 (Add)**: 새로운 게임을 리스트에 등록합니다. VNDB 연동 또는 로컬 이미지 사용이 가능합니다.
- **편집 (Edit)**: 선택한 게임의 제목, 점수, 코멘트 등을 수정합니다.
- **삭제 (Delete)**: 선택한 게임을 리스트에서 영구히 삭제합니다.
- **이미지 내보내기 (Export Image)**: 현재 티어표를 이미지 파일로 저장합니다.
- **데이터 백업 (Backup)**: 현재 모든 데이터를 ZIP 파일로 저장합니다.
- **데이터 복원 (Restore)**: 백업된 ZIP 파일을 불러와 데이터를 복구합니다.
- **팔레트 (Palette)**: 색상 커스텀 모드를 활성화합니다. (좌클릭: 변경 / 우클릭: 초기화)
- **테마 전환 (Theme)**: 다크 모드와 라이트 모드를 전환합니다.
- **설정 (Settings)**: 언어 설정 및 저작권 정보를 확인합니다.
- **GitHub**: 공식 저장소 웹페이지를 방문합니다.

## [ENG] Key Features and Usage

### 🚀 Key Features
- **VNDB API Integration**: Automatically fetch game info and cover images using just a title or ID.
- **Multi-language Support**: Supports 5 languages with seamless switching in settings.
- **Custom Design**: Dark/Light mode support and customizable tier/background colors.
- **Image Export**: Save your tier list as a high-quality JPEG to share.
- **Data Management**: Keep your data safe with built-in backup and restore features.

### 🔘 Button Guide
- **Add**: Register a new game. Supports VNDB fetching or local images.
- **Edit**: Modify the title, score, or comments of the selected game.
- **Delete**: Permanently remove selected games from the list.
- **Export Image**: Save the current tier list as an image file.
- **Backup**: Save all current data as a ZIP file.
- **Restore**: Load a backup ZIP file to recover your data.
- **Palette**: Toggle color customization mode. (Left-click: Change / Right-click: Reset)
- **Theme Toggle**: Switch between Dark and Light modes.
- **Settings**: Change app language and view copyright info.
- **GitHub**: Visit the official repository.

## [JPN] 主な機能と使用方法

### 🚀 主な機能
- **VNDB API 連携**: タイトルやIDだけでゲーム情報とカバー画像を自動で取得します。
- **多言語対応**: 5カ国語のUIをサポートし、設定で自由に変更可能です。
- **カスタムデザイン**: ダーク/ライトモード対応、ティアごとの色や背景色を自由に指定できます。
- **画像出力**: 作成したティアリストを高画質JPEG画像として保存し、共有できます。
- **データ管理**: バックアップと復元機能により、データを安全に管理できます。

### 🔘 ボタンの説明
- **追加 (Add)**: 新しいゲームをリストに登録します。VNDB連携またはローカル画像が使用可能です。
- **編集 (Edit)**: 選択したゲームのタイトル、スコア、コメントなどを修正します。
- **削除 (Delete)**: 選択したゲームをリストから永久に削除します。
- **画像出力 (Export Image)**: 現在のティアリストを画像ファイルとして保存します。
- **バックアップ (Backup)**: 現在のすべてのデータをZIPファイルとして保存します。
- **復元 (Restore)**: バックアップされたZIPファイルを読み込み、データを復旧します。
- **パレット (Palette)**: 色のカスタマイズモードを有効にします。（左クリック：変更 / 右クリック：初期化）
- **テーマ切替 (Theme)**: ダークモードとライトモードを切り替えます。
- **設定 (Settings)**: 言語設定および著作権情報を確認します。
- **GitHub**: 公式リポジトリのウェブページにアクセスします。

## [CHN] 主要功能与使用方法

### 🚀 主要功能
- **VNDB API 联动**: 仅需标题或ID即可自动获取游戏信息和封面图片。
- **多语言支持**: 支持5种语言界面，可在设置中自由切换。
- **自定义设计**: 支持深色/浅色模式，可自定义每个等级的颜色及背景色。
- **导出图片**: 将制作好的等级表保存为高质量JPEG图片进行分享。
- **数据管理**: 通过备份和还原功能安全地管理您的数据。

### 🔘 按钮说明
- **添加 (Add)**: 将新游戏添加到列表中。支持VNDB获取或使用本地图片。
- **编辑 (Edit)**: 修改所选游戏的标题、评分、评论等信息。
- **删除 (Delete)**: 从列表中永久删除选中的游戏。
- **导出图片 (Export Image)**: 将当前等级表保存为图片文件。
- **数据备份 (Backup)**: 将当前所有数据保存为ZIP文件。
- **数据还原 (Restore)**: 加载备份的ZIP文件以恢复数据。
- **调色板 (Palette)**: 激活颜色自定义模式。（左键：更改 / 右键：重置）
- **主题切换 (Theme)**: 在深色模式和浅色模式之间切换。
- **设置 (Settings)**: 更改语言设置并查看版权信息。
- **GitHub**: 访问官方仓库网页。

## [ESP] Funciones Principales y Uso

### 🚀 Funciones Principales
- **Integración con VNDB API**: Obtén automáticamente info del juego e imágenes de portada con solo el título o ID.
- **Soporte Multilingüe**: Soporta 5 idiomas con cambio instantáneo en los ajustes.
- **Diseño Personalizado**: Soporte para modo oscuro/claro y colores de tier/fondo personalizables.
- **Exportar Imagen**: Guarda tu lista de tiers como una imagen JPEG de alta calidad para compartir.
- **Gestión de Datos**: Mantén tus datos seguros con funciones de respaldo y restauración.

### 🔘 Guía de Botones
- **Añadir (Add)**: Registra un nuevo juego. Permite usar VNDB o imágenes locales.
- **Editar (Edit)**: Modifica el título, puntuación o comentarios del juego seleccionado.
- **Eliminar (Delete)**: Elimina permanentemente los juegos seleccionados de la lista.
- **Exportar Imagen (Export Image)**: Guarda la lista de tiers actual como un archivo de imagen.
- **Respaldo (Backup)**: Guarda todos los datos actuales en un archivo ZIP.
- **Restaurar (Restore)**: Carga un archivo ZIP de respaldo para recuperar tus datos.
- **Paleta (Palette)**: Activa el modo de personalización de colores. (Click izquierdo: Cambiar / Click derecho: Restablecer)
- **Cambiar Tema (Theme)**: Alterna entre el modo Oscuro y Claro.
- **Ajustes (Settings)**: Cambia el idioma de la aplicación y consulta la información de copyright.
- **GitHub**: Visita el repositorio oficial.
