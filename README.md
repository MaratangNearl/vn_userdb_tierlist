# 비주얼 노벨 티어리스트 앱

데스크탑 환경에서 비주얼 노벨 티어리스트를 관리하고 이미지로 내보낼 수 있는 프로그램입니다.
(본 애플리케이션은 Gemini 3.1 Pro를 활용하여 제작되었습니다.)

## 기능
- VNDB API 연동을 통한 게임 표지 및 정보 자동 입력
- 점수대(10점 단위)별 티어리스트 동적 생성
- 티어리스트 전체 이미지 내보내기 (JPG 1200px 폭)
- 데이터 백업 및 복원 기능

## 설치 방법
1. Python 3.10 이상이 설치되어 있어야 합니다.
2. 패키지 의존성을 설치합니다.
```bash
pip install -r requirements.txt
```

## 실행 방법
### 실행 파일로 실행 (권장)
가장 쉬운 방법은 미리 빌드된 실행 파일을 사용하는 것입니다.
`dist/VNTierList/VNTierList.exe` 파일을 더블클릭하여 실행하세요.
(`assets` 폴더와 함께 배포되는 `VNTierList` 폴더 전체를 유지해야 정상 동작합니다.)

### 파이썬 코드로 직접 실행
파이썬 환경이 구축된 경우 소스코드로도 실행 가능합니다.
```bash
python main.py
```
